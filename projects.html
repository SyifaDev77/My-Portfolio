Projects

This section highlights selected projects that demonstrate my computer science background alongside my expertise in multimedia computing, interactive systems, and application development.

Research Ideation Canvas - GERIC

An interactive web-based system collaboratively developed during my internship to organize and visualize research ideas using structured frameworks and collaborative ideation tools.

GIMS - GERIC Integrated Management System

A web-based management system developed during my internship to support GERIC’s internal operations by centralizing data management, reporting, and administrative workflows into a single integrated platform.

AR Volcano

ARVolcano App was developed for my Final Year Project (FYP) with the goal of enhancing the learning experience in earth science education. The application uses augmented reality to project a 3D volcano model into the user’s environment, allowing them to rotate, zoom, and explore different volcanic layers. It also includes animations that simulate eruptions and interactive information panels to help users understand how volcanoes work.

Jungle Paws Web Based System

Jungle Paws is a web-based booking system, designed specifically for managing reservations at a cat hotel. The system allows users to create an account, update their profile, and browse available cat-care packages before making a booking. Once a reservation is completed, the system automatically generates a printable receipt for record-keeping.

Sign Language Detection App

This application was developed as part of a course assignment to demonstrate the implementation of Artificial Intelligence (AI) algorithms through machine learning techniques for sign language gesture detection and interpretation.

Math Bingo Game App

An educational game application designed to make learning basic mathematics more engaging through interactive bingo-style gameplay. The app reinforces arithmetic skills while providing a fun and user-friendly experience.

Blood Bank Hospital System

A web-based system designed to manage blood donation and inventory records for hospital use. The system supports efficient tracking of donor information, blood stock levels, and basic administrative workflows to improve data organization and accessibility.

Froggy Forest - Game

A 2D casual game set in a forest environment, designed to provide a light and engaging gameplay experience. Players control a frog character to navigate obstacles and challenges, focusing on simple mechanics and enjoyable interaction.

RoomCraft - Web Based

A web-based application that allows users to design and visualize room layouts interactively. The system helps users experiment with space planning and interior arrangements through an intuitive and user-friendly interface.

Soil Monitoring System - ARDUINO

An IoT-based system developed using Arduino to monitor soil conditions such as moisture levels. The system collects sensor data in real time to support basic analysis and decision-making for plant care and agriculture use.

Data Mining

A course assignment that applied data mining techniques using WEKA to analyze datasets and extract meaningful patterns and insights. The project demonstrates the process of preprocessing, analyzing, and interpreting data for decision-making purposes.

Scratch - Game

A simple interactive game developed using Scratch to demonstrate programming logic, event handling, and basic game design concepts. The project emphasizes learning through play and creative problem-solving.

Audio Compression

A project focused on reducing audio file sizes while maintaining sound quality. Techniques in digital signal processing were applied to analyze and compress audio data efficiently, demonstrating understanding of multimedia and data optimization concepts.

Eon-XR - Meta Experience

An interactive XR experience created using the EON-XR application as part of a course assignment. The project allowed me to design engaging 3D and 360° content and explore immersive user experiences across mobile and VR devices.

Blender Project

3D objects modeled and designed using Blender to demonstrate proficiency in 3D modeling, texturing, and visualization. The project showcases creativity and understanding of spatial design principles.
