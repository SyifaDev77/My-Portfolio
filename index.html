Projects

AR Volcano

ARVolcano App was developed for my Final Year Project (FYP) with the goal of enhancing the learning experience in earth science education. The application uses augmented reality to project a 3D volcano model into the user’s environment, allowing them to rotate, zoom, and explore different volcanic layers. It also includes animations that simulate eruptions and interactive information panels to help users understand how volcanoes work.

Jungle Paws Web Based System

Jungle Paws is a web-based booking system,designed specifically for managing reservations at a cat hotel. The system allows users to create an account, update their profile, and browse available cat-care packages before making a booking. Once a reservation is completed, the system automatically generates a printable receipt for record-keeping.

Sign Language Detection App

This application was developed as part of a course assignment to demonstrate the implementation of Artificial Intelligence (AI) algorithms through machine learning techniques for sign language gesture detection and interpretation.
